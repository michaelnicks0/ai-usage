AI Usage Tool — Architecture

get-data: normalized balance + token usage across DeepSeek, xAI, and Vast.ai

5 Common Data Points

  • • Account Balance ($)
  • • Tokens In (uncached prompt)
  • • Tokens In (Cached prompt hits)
  • • Tokens Out (completions)
  • • Tokens Total (summed)

Authentication

  • • Reads from ~/.hermes/.env
  • • DeepSeek: API key + platform auth token
  • • xAI: separate management API key
  • • Vast.ai: API key from env or config file

Design Decisions

  • • Python stdlib only — zero dependencies
  • • Fail-open on any provider error
  • • Table + JSON output modes
  • • Filterable by provider (--provider)
