AI Usage Tool — Data Architecture

How raw API fields map to normalized output across all 6 providers

DeepSeek

  • • balance ← total_balance (USD)
  • • tokens_in_hit ← PROMPT_CACHE_HIT_TOKEN
  • • tokens_in_miss ← PROMPT_CACHE_MISS_TOKEN
  • • tokens_out ← RESPONSE_TOKEN
  • • period_spend ← tokens × pricing calc

xAI

  • • balance ← total.val / 100
  • • tokens_in_hit ← Cached prompt text tokens
  • • tokens_in_miss ← Prompt text tokens
  • • tokens_out ← Completion text tokens
  • • period_spend ← totalWithCorr.val / 100

Vast.ai

  • • balance ← credit
  • • period_spend ← sum charges (current month)
  • • tokens — N/A (GPU rental, not LLM)

Exa

  • • balance ← orbCreditsInCents / 100
  • • period_spend ← total_cost_usd
  • • Requires session token for balance
  • • Service key for usage API

X API

  • • balance ← credits.balance
  • • period_spend ← usage × eventTypePricing
  • • Requires auth_token + ct0 cookie
  • • -m flag: per-event-type breakdown
